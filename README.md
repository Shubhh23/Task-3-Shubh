# SQL Data Analysis Project

## Project Overview
This project focuses on analyzing an e-commerce dataset using SQL queries in Jupyter Notebook with SQLite. The dataset was loaded into a SQL database and analyzed using various SQL operations to extract meaningful business insights.

## SQL Concepts Used
- SELECT
- COUNT()
- AVG()
- SUM()
- GROUP BY
- ORDER BY
- WHERE
- Aliases (AS)

## Key Insights
- Total customer orders: 1200
- Average order value: ₹1053.97
- Total revenue: ₹12,64,761.96
- Printer is the most ordered product.
- Phone is the least ordered product.
- Online payment is the most preferred payment method.
- Delivered orders represent successful customer transactions.

## Tools Used
- Python
- Jupyter Notebook
- SQLite
- SQL
- Pandas

## Outcome
Successfully performed SQL-based data analysis and generated business insights from the dataset.
